Mobile Application Development Tool

Tool Name

Android Studio

Version: 3.0.1 for Windows

Platform

Android OS

Application Development Type

  • Used to create native applications for the Android OS

Advantages:

  • Full feature set of the Android OS is available
  • Can target any version of Android OS
  • Applications are written in the native programming language of the operating system (java)

Disadvantages:

  • No cross platform support - can only be used for Android applications
  • Programming language is not transferable to other mobile development

Requirements

  • Microsoft Windows 7/8/10 (32-bit or 64-bit)
  • 2 GB RAM minimum, 8 GB RAM recommended
  • 2 GB of available disk space minimum, 4 GB Recommended (500 MB for IDE + 1.5 GB for Android SDK and emulator system image)
  • 1280 x 800 minimum screen resolution
  • JDK 8
  • For accelerated emulator: 64-bit operating system and Intel processor with support for Intel VT-x, Intel EM64T (Intel 64), and Execute Disable (XD) Bit functionality

IDE

Yes - Android Studio is a fully featured IDE including a code editor and debugger

Required Software

Android Studio requires the JDK (Java Development Kit)

Advantages:

  • Supported by Google - the creators of Android OS
  • Thorough documentation due to large user base
  • Massive feature set, customization

Disadvantages:

  • No cross platform support - can only be used for Android applications
  • Interface can be overwhelming to new users

Language Availability

Most languages are supported, English, French, Russian, German

Uploading and selling an app

A one time fee of $25 is required to publish to the Google Play store. Android Development Studio does not have any fees associated with it

Third Party Emulators

H.a.X.M (Hardware Accelerated Execution Manager)

A plugin for Android Studio which allows hardware level acceleration for Android SDK emulators on Intel Systems. Uses Intel Virtualization Technology.
